# ODI

미니멀한 미술작품 오디오 가이드 서비스

## 기술 스택

- React Native + Expo SDK 54
- TypeScript
- Expo Router

## 빠른 시작

## 실행 방법

| 방법           | 명령어        | 설명             |
| -------------- | ------------- | ---------------- |
| Expo Go        | `npm start`   | QR 코드로 스캔   |
| 웹 브라우저    | `npm run web` | 웹에서 실행      |
| iOS 시뮬레이터 | `npm run ios` | Mac + Xcode 필요 |
