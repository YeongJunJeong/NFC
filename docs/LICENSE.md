## 주의사항

### 1. 라이선스 고지
상업적 사용 시에도 각 라이선스의 고지 사항을 준수해야 합니다:
- MIT: 라이선스 텍스트와 저작권 고지 포함
- Apache 2.0: 라이선스 텍스트, 저작권 고지, 변경사항 명시

### 2. Expo 서비스
- **Expo SDK 자체**: 무료, 상업적 사용 가능
- **Expo 서비스 (EAS Build, EAS Submit 등)**: 유료 플랜 존재 (선택사항)
  - 개발 단계: 무료
  - 프로덕션 빌드: 무료 플랜 있음 (제한적)
  - 대규모 사용: 유료 플랜 필요할 수 있음

### 3. App Store 배포
- iOS App Store: Apple의 정책 준수 필요 (연간 $99 개발자 계정 필요)
- Google Play Store: Google의 정책 준수 필요 (일회성 $25 등록비)

## 권장사항

1. **라이선스 파일 포함**: 프로젝트 루트에 `LICENSE` 파일 추가
2. **라이선스 고지**: 앱 내 또는 문서에 라이선스 정보 포함
3. **의존성 확인**: 새로운 패키지 추가 시 라이선스 확인
4. **법적 검토**: 대규모 상업적 프로젝트의 경우 법무팀 검토 권장
